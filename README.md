# Data Structures and Algorithms - C++ Solutions

![C++](https://img.shields.io/badge/Language-C++-blue) ![DSA](https://img.shields.io/badge/Topic-DSA-orange)

This repository contains my solutions to various DSA problems in C++, organized by topic.  
Each folder contains problem-wise solutions with clean C++ code, designed to strengthen DSA and prepare for coding interviews.

## Folder Structure
- **Linked List/** - Solutions for linked list problems from LeetCode  
- **Array/** - Solutions for array problems  
- **Strings/** - Solutions for string problems  
- **Stacks/** - Solutions for stack problems 
- **Queues/** - Solutions for queue problems 

## How to Use
- Navigate to the topic folder you are interested in.
- Open the `.cpp` files to see the solutions.
- These are primarily designed to run on LeetCode.
